Le plugin Jarvis permet d'ouvrir des dossiers, documents, lancer des logiciels (jeux, editeur de texte...), ouvrir des pages web, agir sur les périphériques du PC. Pour définir cette période, il faut modifier le paramètre ctxTimeout dans le fichier custom.ini se trouvant lui même dans le dossier du plugin : Plugins/jarvis/bin.
Par exemple pour désactiver Sarah pendant trois heures il faudra mettre 10800000 (1000 millisecondes x 60 secondes x 60 minutes x 3 heures).

Les phrases reconnues par Sarah :

  • Sarah, es-tu là ? : elle répond "oui je t'écoute" lorsqu'elle n'est pas en veille, ou sinon "je suis actuellement en veille"
  • Sarah, mets toi en veille : Sarah n'écoute plus aucun ordre durant le temps défini par ctxTimeout
  • Sarah, reviens maintenant : Sarah va dire "je suis de retour" et on peut de nouveau utiliser toutes les commandes vocales
